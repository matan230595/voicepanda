# Pandavoice 🐼

מערכת חכמה ומתקדמת לתמלול דיבור לטקסט בזמן אמת עם יכולות עריכה, ניסוח מחדש בעזרת בינה מלאכותית, תרגום לשפות שונות וסנכרון אוטומטי ומאובטח.

## תכונות עיקריות

- 🎤 **הקלטת קול והמרה לטקסט** - תמלול דיבור בזמן אמת בעברית
- ✨ **עיבוד חכם עם AI** - תיקון, שיפור, סיכום ופרפרזה של טקסט
- 🌐 **תרגום ל-8 שפות** - תרגום מקצועי לעברית, אנגלית, ערבית, רוסית, ספרדית, צרפתית, גרמנית ואיטלקית
- 🔊 **הקראת טקסט (TTS)** - הקראה בקולות שונים עם שליטה במהירות וגובה צליל
- 💾 **שמירה אוטומטית** - כל הנתונים נשמרים אוטומטית בענן
- 📱 **שיתוף מהיר** - שיתוף לווטסאפ, טלגרם וייצוא לקובץ
- 🌙 **מצב לילה** - ממשק מותאם לשימוש במצבי תאורה שונים
- ♿ **נגישות מלאה** - תמיכה בקוראי מסך וניווט במקלדת

## דרישות מערכת

- דפדפן Chrome/Edge (גרסה 33 ומעלה) - נדרש לתכונת הקלטת הקול
- חיבור אינטרנט פעיל - לשימוש בתכונות AI ותרגום
- מיקרופון פעיל - להקלטת קול

## התקנה והרצה

### התקנה מקומית

```bash
# התקנת תלויות
npm install

# הרצת שרת פיתוח
npm run dev

# בנייה לייצור
npm run build
```

### משתני סביבה

המערכת דורשת מפתחות API לשימוש בתכונות מתקדמות. ניתן להגדיר אותם דרך ממשק ההגדרות:

- **Google Gemini** - מומלץ להתחלה (חינמי) - [קבל מפתח](https://makersuite.google.com/app/apikey)
- **Groq** - המהיר בעולם (חינמי) - [קבל מפתח](https://console.groq.com/keys)
- **OpenAI** - החכם ביותר (בתשלום) - [קבל מפתח](https://platform.openai.com/api-keys)

## שימוש בסיסי

### הקלטת קול

1. לחץ על כפתור המיקרופון הגדול במרכז המסך
2. אשר גישה למיקרופון בדפדפן (פעם ראשונה)
3. דבר בקול ברור ורואה את הטקסט מופיע בזמן אמת
4. לחץ שוב לעצירת ההקלטה

### עיבוד עם AI

1. הקלד או הקלט טקסט
2. לחץ על כפתור "AI"
3. בחר פעולה מהירה או כתוב הוראה מותאמת אישית
4. לחץ "שלח" ורואה את הטקסט משתנה

### תרגום

1. הקלד או הקלט טקסט
2. לחץ על כפתור "תרגום"
3. בחר שפת מקור ושפת יעד
4. לחץ "תרגם עכשיו"

### הקראת טקסט

1. הקלד או הקלט טקסט
2. לחץ על כפתור "הקראה"
3. בחר קול, התאם מהירות וגובה צליל
4. לחץ "הקרא טקסט"

## טכנולוגיות

- **Frontend**: React 19, TypeScript, Tailwind CSS, Zustand
- **Backend**: Hono (Cloudflare Workers)
- **AI/ML**: Google Gemini, Groq, OpenAI
- **Speech**: Web Speech API (Recognition & Synthesis)
- **Routing**: React Router v7
- **Build**: Vite
- **Deployment**: Cloudflare Workers

## פתרון בעיות נפוצות

### ההקלטה לא עובדת

- וודא שאתה משתמש בדפדפן Chrome/Edge
- בדוק שנתת הרשאה למיקרופון
- וודא שאתה ב-HTTPS (או localhost לפיתוח)
- נסה לרענן את הדף ולאשר שוב את ההרשאות

### AI לא מגיב

- בדוק שהגדרת מפתח API בהגדרות
- ודא שהמפתח תקף ופעיל
- בדוק שיש לך יתרת API במשתמש שלך
- נסה מפתח API אחר (Gemini/Groq/OpenAI)

### התרגום לא מדויק

- המערכת משתמשת ב-AI לתרגום - התוצאות עשויות להשתנות
- נסה לנסח את הטקסט בצורה פשוטה יותר
- לתוצאות מקצועיות יותר, השתמש ב-OpenAI

### ההקראה לא נשמעת טוב

- נסה קולות שונים - יש עשרות אפשרויות
- התאם את המהירות וגובה הצליל בהגדרות
- סנן לפי שפה מועדפת למציאת קולות איכותיים יותר
- קולות מקומיים (Local) בדרך כלל באיכות טובה יותר
